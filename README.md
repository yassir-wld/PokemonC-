# Pokémon C++ Battle System

1 Project Overview
This is a console-based Pokémon battle simulator developed in C++.  
It demonstrates core Object-Oriented Programming (OOP) principles including:

- Inheritance
- Polymorphism
- Encapsulation
- Abstract classes
- Type advantage system

2 Features
- 4 Pokémon types: Fire, Water, Grass, Electric
- Strength & weakness mechanics
- Turn-based combat system
- Clean class-based architecture

3 Technologies Used
- C++
- OOP Design
- Git & GitHub

4 How to Run
1. Compile the code: `g++ main.cpp -o pokemon`
2. Run the executable: `./pokemon`
